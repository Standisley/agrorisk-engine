# AgroRisk Engine - Skills

Esta pasta contém as definições das ferramentas e lógicas determinísticas que validam as regras de negócio do motor de crédito. O Agente Antigravity utiliza os módulos e funções deste diretório para auditar, checar elegibilidade (como a função de validação de zoneamento agrícola ZARC) e processar as políticas de tomada de decisão.
